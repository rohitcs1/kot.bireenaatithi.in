## âœ¨ Key Features

1. **Real-Time Sync**: Categories added in frontend immediately reflect in database
2. **Multi-Hotel Support**: Each hotel sees only its own categories
3. **Role-Based Access**: Only admin/manager can create categories
4. **JWT Authentication**: All requests require valid Bearer token
5. **Error Handling**: Meaningful error messages on frontend
6. **Auto-Retry**: 401 errors redirect to login automatically
7. **Validation**: Both frontend and backend validate inputs
8. **Data Integrity**: Foreign key constraints ensure data consistency
